# Homebrew Tap for Diffy

This is the Homebrew tap for [Diffy](https://github.com/atlascodesai/diffy), a modern, native diff tool for macOS.

## Installation

```bash
brew tap atlascodesai/diffy
brew install diffy
```

Or install directly:

```bash
brew install atlascodesai/diffy/diffy
```

## Usage

After installation, you can:

**Open the app:**
```bash
open -a Diffy
```

**Run from terminal:**
```bash
diffy
```

## Updating

```bash
brew update
brew upgrade diffy
```

## Uninstalling

```bash
brew uninstall diffy
brew untap atlascodesai/diffy
```
